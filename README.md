# SCT-Cipher 加密算法技术文档

## 算法概述
一种结合前沿密码学机制的**对称加密算法**，具备抗量子计算特性，适用于最高安全等级的数据保护场景。

---

## 对称加密验证矩阵

| 验证指标             | 本算法实现方案                   |
|----------------------|--------------------------------|
| 密钥模式             | 256/512-bit共享密钥            |
| 加密效率             | ≥600MB/s（AES-NI加速时）       |
| 加密可逆性           | 需完全对称的密钥与操作逻辑      |
| 抗量子性             | 512位抗碰撞保障                |
| 标准化兼容           | NIST SP 800-185核心组件继承     |
